Remote Sensing Image Classification

The goal of this Platform is to outline possible approaches to retrieve a land-use / land cover (LULC) classification based on Optical and RADAR satellite image products. It provides infographics on image selection as well as highlighting the difference between supervised and unsupervised classification on Optical Image products. The Infographics presented in this platform are suggestions on how to satellite imagery can be transformed into a land-use / land cover products, but as always, there are other ways in which this can be achieved.

Photo by USGS on Unsplash

Imagery Classification

Is the process of extracting thematic information from a raster data about a particular feature or Area. This information which is usually in the form of Land cover classes is useful in urban planning, resource management, change detection, agriculture etc.

Types of Image Classification

Image Classication can be Pixel-based which considers each pixel individually, ignoring the neighboring pixels. This can be used in both supervised and unsupervised methods, OR Object-based which uses a combination of adjacent values and spectral characteristics to build objects (segments).

OPTICAL Image Classification

The datasets used in this section of the infographics are a part of a Landsat 8 OLI IW product covering the Victoria Falls in Zambia for 05-09-2015, 13-09-2018 and 05-09-2021. After preprocessing, the Random Forest classifier (Supervised Classification) and Iso Cluster Unsupervised Classification were applied to generate LULC image products.
